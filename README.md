

## 按键功能及操作

#### 适用型号：K388,H5,S6,X8

|按键|操作|功能|
|:--:|:--:|:--:|
|音量上键|单击|返回|
|音量上键|双击|多任务|
|音量上键|长按|主界面|
|音量下键|双击|  通知栏  |
|音量下键|长按|学习桌面|

#### 适用型号：X5pro,Y29,K21

|   按键   | 操作 |   功能   |
| :------: | :--: | :------: |
|  返回键  | 单击 |   返回   |
|  返回键  | 双击 |  多任务  |
|  返回键  | 长按 |  主界面  |
| 音量上键 | 长按 |  通知栏  |
| 音量下键 | 长按 | 学习桌面 |



## 提醒

1. 融过卡的机器不要在开机状态下拔卡：可能造成系统损坏。
2. 不要恢复出厂设置：原来的软件会消失。
误操作的后果自负。
